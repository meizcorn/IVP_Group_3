# IVP_Group_3
Introduction to Image &amp; Video Processing group project for Group 3.

There are 2 branches for this project; main and 'Final_Test_Evaluation'. All model pipelines from build to train and 
evaluation, the preprocessing steps and the final csv submission (done with the HOG-SVM model) can be 
found on the main branch.

Comparisons of the two models can be found on the 'Final_Test_Evaluation' branch.

All group members have commited in this project, two members worked on the main branch and the third member worked on
the 'Final_Test_Evaluation' branch.
